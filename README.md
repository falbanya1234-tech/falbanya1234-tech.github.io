# falbanya1234-tech.github.io

えりにゃんアプリ の開発者向け GitHub Pages サイトです。

## 公開 URL

https://falbanya1234-tech.github.io/

## このリポジトリについて

Android アプリ「ひびカレ」の Google Play 公開に必要な  
developer website として使用しています。

| ファイル | 内容 |
|----------|------|
| `index.html` | 開発者サイトのトップページ |
| `privacy-policy.html` | プライバシーポリシー |
| `app-ads.txt` | AdMob 広告用（AdMob 登録後に追加予定） |

## app-ads.txt について

AdMob の Publisher ID 取得後に以下の形式で追加します。

```
google.com, pub-XXXXXXXXXXXXXXXXX, DIRECT, f08c47fec0942fa0
```

Publisher ID は AdMob コンソールのアカウント情報ページで確認できます。

## GitHub Pages の有効化

リポジトリの Settings > Pages > Source を  
`Deploy from a branch` → `main` / `root` に設定してください。
